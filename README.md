# GitHub Follow2Follow

A C# program to check the "follow-back" status on GitHub by scraping a user's followers and following lists. The program finds users that a specific user is following but who are not following back, without using the GitHub API.

## Features

- Collects followers and following lists of a GitHub user.
- Identifies users who are not following back.
- Works without GitHub API or authentication tokens.

## Requirements

- [.NET Core SDK](https://dotnet.microsoft.com/download) (version 5.0 or later)
- [HtmlAgilityPack](https://html-agility-pack.net/) library for HTML parsing
